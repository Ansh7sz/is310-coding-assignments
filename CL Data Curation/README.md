# Pokédex CLI Data Entry

This is a simple command-line application written in Python that lets you add Pokémon data and save it to a CSV file.

## Features

- Displays example Pokémon in a formatted table.
- Optionally shows previously saved Pokémon from a CSV file.
- Lets you enter multiple Pokémon in one session.
- Confirms each entry before saving it.
- Saves all confirmed entries to `output/pokemon.csv`.
- Prints the full path to the CSV file so you can find it easily.

Each Pokémon entry includes:
- Name  
- Generation number  
- Primary type  
- Secondary type (optional)
